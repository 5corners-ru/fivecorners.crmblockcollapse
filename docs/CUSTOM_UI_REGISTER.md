# CUSTOM_UI_REGISTER — зарегистрированные отступления от UI-кита

Канон требует: любое отклонение от UI-кита `fivecorners.admintemplate` фиксируется здесь
с обоснованием и условием пересмотра. Молчаливых отступлений быть не должно.

## 1. Шапка admin-страницы — класс `PageHeader` вместо `DemoPageHelper`

- **Что:** единая шапка admin-страницы реализована классом `lib/PageHeader.php`, а не
  скопированным из эталона `DemoPageHelper`.
- **Статус канона:** функционально соответствует. Рендерит ту же каноничную шапку
  `.fco-custom-header` — версия + системный ID справа, иконка-наушники (чат открытой линии
  5 УГЛОВ через `CRM_WIDGET_URL` в `AdminMenu::onProlog`), брендовая ссылка «5 УГЛОВ».
- **Отличие:** нет константы `NAV_PAGES` и навигации по разделам — у модуля **одна**
  admin-страница, навигировать не между чем.
- **Обоснование:** переписывание в `DemoPageHelper` ради имени класса — churn без изменения
  того, что видит пользователь (рендер идентичен). Сохранены каноничные хуки (наушники в
  `renderClose`, loader виджета под guard'ом префикса `fc_crmblockcollapse_` в `onProlog`).
- **Условие пересмотра:** появится вторая admin-страница → завести `NAV_PAGES`-навигацию
  (либо мигрировать на `DemoPageHelper`).

## 2. Внешний Google Fonts (Open Sans) в шапке

- **Что:** `PageHeader::addStyles()` грузит `fonts.googleapis.com` / `fonts.gstatic.com`.
- **Статус канона:** соответствует эталону — `DemoPageHelper` грузит шрифт так же.
- **Риск:** на air-gapped / сетево-ограниченных порталах шрифт не подгрузится (фолбэк на
  системный), плюс referrer-утечка факта захода. Не исполняемый JS — не RSI.
- **Условие пересмотра:** запрос от клиента на air-gap-совместимость → самохостинг subset
  (`install/js/fonts/`). Заведено как [BACKLOG TD-3](../BACKLOG.md).

## 3. CRM-виджет (наушники) грузит внешний JS с portal.5corners.ru

- **Что:** `AdminMenu::onProlog` инжектит loader `https://portal.5corners.ru/.../loader_5_l7a35t.js`
  на admin-страницах модуля.
- **Статус канона:** **это рабочая фича по канону** (MIGRATION.md §3.3) — чат открытой линии
  5 УГЛОВ за иконкой-наушники, не «мёртвый код» и не самовольное RSI.
- **Оговорка:** для публикации в Marketplace remote-JS режут — там виджет надо убрать/локализовать.
