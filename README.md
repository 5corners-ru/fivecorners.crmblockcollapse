# fivecorners.crmblockcollapse

Модуль Bitrix24 (D7, on-premise) — добавляет кнопки **сворачивания/разворачивания блоков**
на карточках CRM-сущностей (Сделки, Лиды, Контакты, Компании, Смарт-процессы). Состояние
сворачивания запоминается **per-user** и восстанавливается между сессиями. Дополнительно —
правила автоматического раскрытия блоков **по стадии** сущности.

## Назначение

В карточке CRM много секций (`ui-entity-editor-section`). Модуль вешает на заголовок каждой
секции кнопку-«галочку» и кликабельный заголовок: клик сворачивает/разворачивает тело секции
с анимацией. Выбор пользователя сохраняется в `b_user_option` и применяется при следующем
открытии карточки. Админ может задать правила «на стадии X блок Y всегда раскрыт».

## Два сценария

1. **Рядовой пользователь.** Открывает карточку сделки → видит кнопки сворачивания на секциях →
   сворачивает ненужное → при следующем заходе всё восстановлено как оставил. При смене стадии
   сделки блоки из правил стадии раскрываются автоматически.
2. **Администратор.** Идёт в «5 УГЛОВ → Сворачивание блоков в CRM»
   (`/local/admin/fc_crmblockcollapse_settings.php`) → включает нужные типы сущностей и
   смарт-процессы → задаёт правила раскрытия по стадиям.

## Quickstart

1. Установить модуль (Marketplace → Установленные решения, или копированием в `/local/modules/`).
2. Открыть страницу настроек, включить типы сущностей (по умолчанию включены все).
3. Открыть любую карточку CRM — кнопки сворачивания уже работают.
